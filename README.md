## 📖 이번 편에 담긴 진짜 계량분석 결과
- 은행 **12곳** 조사 → 합산 **288조원**어치 갑옷(위험가중자산)이 풀림
- **튼튼함 점수(보통주자본비율 CET1) 평균 +3.2점**
- 가장 많이 푼 은행: **기업은행 53조** · 신한 49조 · 산업 40조 · 국민 38조 · 하나 38조
- 자산별 이질성: 기업 **+30%**, 기타소매 +30%, 특수금융 +14%, **집담보 −9%(효과 없어 제외)**
- **금융감독원 장부(FISIS)로 교차검증** → 숫자가 거의 그대로 일치

## 📏 잣대 이야기 (표준법 vs 내부등급법)
- **공용 잣대 = 표준법(SEC-SA)**: 누구나 공개 장부로 잴 수 있음 → 이 책이 쓴 방법
- **정밀 잣대 = 내부등급법(SEC-IRBA)**: 더 딱 맞지만 **은행 내부 비밀 숫자(부도확률 등)가 필요**해 공개자료만으로는 어려움

## 🔑 비유 풀이
| 동화 속 | 진짜 뜻 |
|---|---|
| 곰돌이 / 갑옷 | 은행 / 위험가중자산(RWA) |
| 위험 도깨비 | 대출의 신용위험 |
| 기사님 / 작은 선물 | 투자자 / 보증수수료 |
| 부엉이 박사 | 연구자 |
| 튼튼함 점수 | 보통주자본비율(CET1) |
| 금융감독원 장부 | 금융통계정보시스템(FISIS) |
| 공용/정밀 잣대 | 표준법(SEC-SA)/내부등급법(SEC-IRBA) |
